### ✨ ویژگی‌های کلیدی

- 🛡️ **ایمنی فوق‌العاده بالا:** استفاده از زنجیره اختصاصی (`RCL_IPTABLE`). تضمین ۱۰۰٪ برای عدم تداخل با قوانین Docker، KVM، Virtualizor و فایروال پیش‌فرض.
- 🌐 **مسیریابی چندگانه (Multi-Server):** فوروارد پورت‌های محلی مختلف به آی‌پی‌ها و پورت‌های مقصد متفاوت با پشتیبانی از `TCP`، `UDP` و `TCP+UDP` همزمان.
- 🤖 **کشف هوشمند شبکه:** تشخیص خودکار اینترفیس شبکه اصلی و MTU بدون نیاز به تنظیمات دستی کاربر.
- ⚡ **بهینه‌سازی شبکه (یک‌کلیکی):** فعال‌سازی و غیرفعال‌سازی آسان پروتکل‌های `BBR`، `FQ Queue` و `MSS Clamping` برای افزایش سرعت و پایداری تانل.
- 📡 **مانیتورینگ زنده (Live Monitoring):** مشاهده لحظه‌ای ترافیک ورودی (پکت‌ها و بایت‌ها) برای هر روت با رفرش خودکار.
- 🩺 **چک‌سلامت (Health Check):** بررسی وضعیت قوانین DNAT/FORWARD و دسترسی‌پذیری سرور مقصد.
- 🔄 **پایداری مطلق (Persistence):** ذخیره خودکار قوانین پس از هر تغییر تا در ریبوت‌های سرور هیچ تغییری ایجاد نشود.
- 💾 **پشتیبان‌گیری و بازیابی:** بک‌آپ‌گیری فایل‌های تنظیمات، دیتابیس روت‌ها و قوانین فایروال با قابلیت بازگردانی سریع.
- 🎯 **جلوگیری از تداخل (Conflict Detection):** هوشمندسازی در زمان تعریف پورت‌های تکراری با پیشنهاد جایگزینی.

### ⬇️ نصب و راه‌اندازی

نصب بسیار ساده است و اسکریپت به صورت خودکار تبدیل به یک دستور جهانی در سرور شما می‌شود. 

کافیست دستورات زیر را به ترتیب در ترمینال با دسترسی `root` اجرا کنید:

```bash
# 1. دانلود فایل اسکریپت
curl -Ls https://raw.githubusercontent.com/Emmanuel-RCL/iptable-pro/main/iptable-pro.sh -o iptable-pro.sh

# 2. اعمال دسترسی اجرایی
chmod +x iptable-pro.sh

# 3. اجرای اسکریپت (در اولین اجرا به صورت خودکار نصب می‌شود و منو باز می‌گردد)
bash iptable-pro.sh
```

> **نکته:** با اجرای دستور بالا، اسکریپت خود را در مسیر `/usr/local/bin/rcl-iptable` ثبت می‌کند. از این پس می‌توانید در هر کجای سرور فقط با تایپ `rcl-iptable` وارد پنل مدیریت شوید و فایل دانلود شده اولیه را پاک کنید.

### 🎮 نحوه استفاده

پس از نصب، در ترمینال تایپ کنید:
```bash
rcl-iptable
```
منوی تعاملی ظاهر می‌شود که کارهای پیچیده شبکه‌ای را به چند کلیک ساده تبدیل می‌کند. در تمام منوها، گزینه `0` معادل بازگشت (Back) یا خروج (Exit) است.
